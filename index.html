TopRider

Интернет-магазин электровелосипедов, самокатов и байков. Сайт представляет собой каталог товаров с возможностью фильтрации, поиска, добавления в корзину и оформления заказа через WhatsApp. Встроенная админ-панель позволяет управлять контентом.

Основные возможности

  • Каталог товаров: Просмотр товаров с фото, ценами и характеристиками.
  • Умный поиск: Поиск по названию товара.
  • Фильтрация: По категориям, цене, бренду, размеру рамы, мощности мотора и другим параметрам.
  • Корзина: Сохранение товаров в LocalStorage браузера.
  • Быстрый просмотр: Всплывающее окно с детальной информацией.
  • Оформление заказа: Генерация сообщения и отправка менеджеру в WhatsApp.
  • Адаптивность: Полная поддержка мобильных устройств.
  • Интеграции: Ссылки на социальные сети и 2GIS.
Stack: HTML  |  Styling: Tailwind CSS  |  Logic: JavaScript (Vanilla)  |  Storage: LocalStorage
